# DevOps AWS Project

## Project Structure
- **/terraform**: Infrastructure as Code for AWS.
- **/docker**: Application containerization settings.
- **/scripts**: Automation and monitoring scripts.

## How to use
1. **Terraform**: Navigate to /terraform, run `terraform init` and `terraform apply`.
2. **Docker**: Navigate to /docker, run `docker-compose up -d`.

## Security
Note: Secrets, SSH keys (*.pem), and Terraform state files are excluded from this repository for security reasons.
